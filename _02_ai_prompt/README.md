## ðŸ›  Core Prompting Techniques (Mandatory)

This module covers the essential strategies required to build robust AI applications.

### 1. Tips and Tricks for Prompt Engineering

Learn the foundational "Dos and Don'ts," including being specific, providing context, and defining the desired output format to reduce hallucinations.

### 2. Chain of Thoughts (CoT) Technique

An explanation of the "Chain of Thoughts" approach, which encourages the model to show its work. By breaking down complex problems into intermediate steps, the LLM can solve logical tasks more accurately.

### 3. Tree of Thoughts (ToT) Technique

An evolution of CoT, the "Tree of Thoughts" allows the model to explore multiple reasoning paths simultaneously, look ahead, and backtrack when necessary to solve complex problems.

### 4. ReAct Technique

Learn the **Reason + Act** method. This technique combines reasoning traces with task-specific actions, allowing the LLM to interact with external tools or environments to gather information before providing a final answer.
